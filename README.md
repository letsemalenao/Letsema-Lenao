# A Brief Description 
Alpha Dreamers Banking Consortium is experiencing an increase in personal loan defaulters, which negatively affects profitability, operational efficiency, and customer trust. This project applies Exploratory Data Analysis (EDA), data-cleaning techniques, and machine learning, including Logistic Regression, to identify patterns associated with loan default risk.
